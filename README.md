Feedback Management System
Introduction
The Feedback Management System is a decentralized application (dApp) built on the Ethereum blockchain. It enables students to submit feedback which can then be viewed by the admin. The system leverages smart contracts written in Solidity, deployed on the Sepolia test network, and utilizes MetaMask for secure Ethereum transactions.

Features
Student Feedback Submission: Students can submit feedback which is stored on the blockchain.
Admin Feedback Viewing: The admin can view all feedback submissions.
Secure Transactions: Utilizes MetaMask for secure and authenticated transactions on the Ethereum blockchain.
Technologies Used
Solidity: For writing smart contracts.
Ethereum Sepolia Test Network: For deploying and testing smart contracts.
MetaMask: For handling Ethereum accounts and transactions.
Web3.js: For interacting with the Ethereum blockchain from the frontend.
Prerequisites
Node.js and npm installed
MetaMask extension installed on your browser
An Ethereum wallet setup in MetaMask
